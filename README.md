# km-swift-ble

You can control your Keigan Motor via Bluetooth Low Energy using iOS devices.

## Requirement

- Swift4
- iOS10+

## Installation

    $ git clone https://github.com/keigan-motor/km-swift-ble
   
***NOTE***
Carthage and CocoaPods will be supported in the future.
